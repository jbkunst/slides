¿Qué es shiny?

¿Qué es shiny?

  • Es un paquete para construir aplicaciones web usando solamente código R
  • De forma fácil. Sin aprender HTML, Javascript o CSS

¿Cómo luce una shiny-app en código?

library(shiny)

ui <- fluidPage(
   sidebarLayout(
      sidebarPanel(
         sliderInput("nrandom", "N a simular:",
                     min = 1, max = 50, value = 30),
         selectInput("type", "Tipo:", choices = c("p", "l", "b"))
      ),
      mainPanel(plotOutput("outplot"))
   )
)

server <- function(input, output) {
   output$outplot <- renderPlot({
     x <- rnorm(input$nrandom)
     plot(cumsum(x)/seq(length(x)), type = input$type)
   })
}
